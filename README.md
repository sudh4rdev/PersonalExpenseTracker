# PersonalExpenseTracker

Currently, I'm tracking my daily spending on a highly sophisticated app : Notes :stuck_out_tongue:
It's a foolproof 4-step process: 
  * Step 1: Open the app.
  * Step 2: Log the 'crucial' data: Date, item, damage to my wallet (£££) and payment method.
  * Step 3: Perform some highly inaccurate mental maths to update the balance.
  * Step 4: Walk home while listening to the exact same song for the 100th time today.

While this has always worked, it has long been my (adulthood)"dream" to build a super easy app in Python 🐍. This is the inception(yay nice movie!)
